### 任务列表

1. 普通任务列表

	>  语法：**横框+ 空格+方括号（方括号中必须有空格或者x） + 空格+ 文本** 

	- [x] 第一条待办事宜 [ 完成状态 ]
		- [ ] 第一条子条目
		- [x] 第二条子条目
		- [x] 第三条子条目
		- [ ] ~~_作废条目,无需完成_~~

	- [ ] 第二条待办事宜
	- [ ] 第三条待办事宜
